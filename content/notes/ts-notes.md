TypeScript is built on top of JavaScript (a superset of JavaScript). Any JavaScript code is valid in TypeScript.

## Variable Type

### any

A variable gets type `any` if it is not initialized when declared.

### Arrays

```typescript
let numbers: numbers[] = [1, 2, 3];
```
